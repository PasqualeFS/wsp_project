Component 1: On-Scroll Animated Header

Source: Codrops

Target Audience

This style of header is fairly common. I think it works for all audiences. It starts out with a prominent size and then gets out of the way when the user starts to scroll.

Client/User Needs

A header like this is nice for minimizing elements that are not part of the main focus. The navigation, logo and some contact information is still readily available and easily accessible for the user. Once the user scrolls down a bit, the header compresses down and it comes as an afterthought to the larger elements clients want site visitors to focus on within the main content space.

Critique

It's clean, it's simple. Just over 10 lines of HTML code, followed by some extensive CSS and a light amount of Javascript. By modifying the colors, fonts and logo area, it can easily be branded for many clients.

Component 2: Responsive, Flat Navigation Menu

Source: CSS Menu Maker

Target Audience

Here's a standalone navigation component that probably works well for creative settings, light, whimsical and the little flare out underline as the user hovers can make it appealing to a more youthful audience. Great use for those browsing the web for entertainment websites, blogs and agency-type businesses.

Client/User Needs

A prominent navigation menu that utilizes hover effects to show the user what they're about to select. The solid underline and font-color change on the active state lets them know which page they're currently visiting.

Critique

I think it's a simple navigation menu. It's light and easy to code. It does not have a sub-menu option built in so larger, professional websites with a good deal of subpages may want to utilize a different approach to their navigation altogether.

Component 3: Responsive Full-Screen Background Image

Source: Web Library

Target Audience

The full-screen background can be used for just about any target audience because the targeting is set by the image used. Be it professional business, a surf shop, a marketing agency, or a photographer website, it can be used to reach a wide range of audiences.

Client/User Needs

It's a way to make a statement. Big, bold and eye catching. It give the client a massive display on large screens that fully shrinks down for various mobile devices. This is a great thing for users on any device.

Critique

I'm still blown away by most websites where I see full-screen background images and/or videos. Some of them are quite breathtaking. When paired with eye-catching fonts and copy that draws interest, it beckons the user to explore further.

Component 4: Social Media Sharing Menu Component

Source: Six Revisions

Target Audience

This component has applications with almost all audiences. It's easier to say you wouldn't use it on pages within a private member section of a website that require a login.

Client/User Needs

Free marketing for the client for users that know someone who has a need to see what the client is promoting. With the onset and massive increase of social sharing and over-sharing, it's safe to say that some form of a sharing option needs to be on nearly all websites.

Critique

I'm not sure this is the best looking social share component but it has simple code and should be able to be styled differently than the demo. Like many of the other components I've chosen, it can take on the look/feel of a client's branding with some minor CSS tweaks. Icons can be swapped for the most current ones. This can also easily be modified to include more or less sharing options for other channels such as Instagram, Pinterest, TripAdvisor and more.

Component 5: Slide-In Image Box Component

Source: CSS-Tricks

Target Audience

This component lends itself well to users seeking articles, tutorials, blog posts, news updates and more. The audience for a component like this can include a wide range of age groups when themed appropriately.

Client/User Needs

Clients can find creative ways to display a snippet of content to tease the user in to wanting to see more. It could be applied toward restaurants, hotels and a wide range of online marketing needs.

Critique

I truly love this. I might like to slow the timing of the animation down that causes the images to fly in from the right. Reading some of the comments on the CSS Tricks page, I see that use of a sprite map was made and this will require some additional research from this article over at A List Apart to fully understand.
